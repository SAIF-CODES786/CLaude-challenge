# Day 4 - Chain-of-Thought Prompting 🧠

## 📌 Overview

Today I explored **Chain-of-Thought Prompting**, an advanced prompting technique that helps AI generate better and more structured responses.

Instead of asking AI for a direct answer, Chain-of-Thought prompting guides AI to analyze the problem step-by-step before creating the final output.

The goal of this task was to create a personalized career roadmap using structured AI reasoning.
